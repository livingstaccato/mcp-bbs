Data Flow:
==========
1. LLM → MCP Client → FastMCP App
2. FastMCP App → TelnetClient → TelnetProtocol
3. TelnetProtocol → Telnet Server → BBS Software
4. BBS → (ANSI Output) → Telnet → Protocol → Terminal Emulator
5. Terminal Emulator → TelnetClient
6. TelnetClient → Auto-Learning → Knowledge Base
7. TelnetClient → Session Logger → session.jsonl
